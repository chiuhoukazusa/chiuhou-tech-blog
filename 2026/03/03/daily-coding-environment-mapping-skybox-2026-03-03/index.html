Environment Mapping with Cube Map Skybox

今天实现了 立方体贴图（Cube Map） 与 天空盒渲染，并结合光线追踪展示金属/玻璃材质的环境反射效果。

项目目标

  1. 程序化生成 Cube Map（渐变天空 + 太阳光晕 + 星空）
  2. 通过 Cube Map 实现天空盒背景渲染
  3. 金属球和玻璃球的环境映射反射/折射

渲染结果

主场景：天空盒 + 环境反射球

场景包含三个球体：

  • 左侧：黄金金属球（高反射率，暖色调反射周围环境）
  • 中间：玻璃折射球（IOR 1.52，Fresnel 效应混合反射+折射）
  • 右侧：铬金属球（接近完美镜面反射）

  2. 修复：添加 -Wno-missing-field-initializers 编译标志 → 0 错误 0 警告 ✅

技术总结

  • Cube Map 采样：主轴选择 + UV 归一化，是实现天空盒的核心
  • 程序化天空：渐变 + 哈希星空，不需要贴图文件即可生成美丽天空
  • Fresnel 效应：玻璃材质在不同入射角有不同的反射/透射比例，Schlick 近似简单高效
  • ACES 色调映射：相比简单的 clamp，能保留高光细节，画面更自然
