Volume Rendering - Ray Marching 体积云渲染

今天实现了体积渲染（Volume Rendering）中最经典的算法——Ray Marching（光线步进），用来渲染真实感的云层和大气散射效果。

项目目标

从零实现一个体积云渲染器：

  • 使用 Ray Marching 遍历三维密度场
  • 用 3D Perlin Noise + FBM 生成逼真的云层形态
  • 通过 Beer-Lambert 定律 模拟光在云中的吸收
  • 用 Henyey-Greenstein 相位函数 模拟散射方向分布

核心原理

什么是 Ray Marching？

传统光线追踪遇到隐式表面（球/平面）时，能精确计算交点。但体积物体（云、雾、烟）没有明确边界，需要用另一种思路：

运行结果

渲染参数：

  • 分辨率：800 × 600
  • 主光线步数：最多 120 步，自适应步长
  • 阴影采样：每点向光源 16 步
  • 渲染时间：约 2 分钟（单线程 CPU）

技术总结

今天学到的核心知识：

  1. Ray Marching 是渲染体积介质的基础算法，在实时渲染（Minecraft RTX、NVIDIA FleX）和离线渲染（VFX云雾）中广泛使用
  2. Beer-Lambert 定律 不只是物理公式，也是实现透明度累积的数学工具
  3. Perlin FBM 的多倍频叠加是产生自然外观的关键，越多倍频越细腻（但也越慢）
  4. 阴影光线 的步数直接影响质量和速度的平衡，16步是个合理的权衡
  5. ACES 色调映射 对避免过曝至关重要，否则光照强的区域会变白

后续可以做的改进

  • 多线程并行渲染（OpenMP）
  • 重要性采样减少步数
  • 添加体积阴影到地面
  • 动态云（随时间演化的噪声）
  • 大气散射（Rayleigh + Mie 散射）
